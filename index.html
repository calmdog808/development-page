Development

Developing (creating menu items) and developing (creating a React App) for the bakery I've always wanted to start

Vision

The goal here was to design an app to show list of items that you could filter and sort while learning React.

I have no React experience and was extremely tired, but if nothing else keeps me going, pastries and an overpriced drink will. So, I chose some basic items (pastries + drinks + ice cream) that you could filter (based on type + dietary restriction) and sort (by price + bestsellers the default) and even add to a favorites list that will keep track of the total of your selections.

Design

Figuring out how to make this.

Key Parts + Considerations

Backend

  • json data for each bakery item
  • BakeryItem component to display an item card for each product
  • State variables to keep track of the current filter state, sort state, and state of the favorites list
  • Functions for both setting the state based off an event and then adjusting which items are shown based on the state
  • Reseting the states except for the favorities when someone clears filters
  • Logic to aggregate the list of favorites and their total price

Frontend

  • MUI cards and grid system to display items with an image, description, and button in an organized manner
  • Separate side bars for filters and the aggregator, so they are not cramped
  • Nav selection (for type) and radio selection (for sort) to select one option bc I wanted to try both types
  • Checkbox selection for dietary restriction because someone can have multiple
  • Add button turns into remove so you can't add duplicates to your favorites list
  • Minimalistic grid layout so the images pop!
